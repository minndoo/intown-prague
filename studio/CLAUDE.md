# Studio

Sanity v5 studio for the InTown project.

## Stack

- Sanity v5
- React 19
- TypeScript 5
- Styled Components 6

## Sanity Project

- Project ID: `hqky80t4`
- Dataset: `production`

## Commands

- `npm run dev` — start dev server
- `npm run build` — build studio
- `npm run deploy` — deploy to Sanity hosting
- `npx eslint .` — lint

## Structure

- `sanity.config.ts` — main config
- `sanity.cli.ts` — CLI config
- `schemaTypes/index.ts` — schema entry point; all types exported here
